# Клиент

При повторном входе зарегистрированные пользователи попадают на экран с приветствием и приглашением войти

<div align="center">
  <img src="/img/intro_auth.png" alt="Страница входа зарегистрированного пользователя" width="300" />
  <figcaption>Для входа в приложения нажмите "Войти"</figcaption>
</div>

После нажатия на кнопку "Войти" пользователи с ролью Клиент попадаю в личный кабинет Клиента

<div align="center">
  <img src="/img/account/client_start.png" alt="Страница входа зарегистрированного пользователя" width="300" />
  <figcaption>Стартовая страница личного кабинета Клиента</figcaption>
</div>

Здесь клиент может перейти сразу в корзину для просмотра и формления заказа
Может перейти в каталог для выбора услуг по уже купленному оборудованию 
Может перейти в каталог для подбора оборудовани и (или) услуг по нему
Может создать свой объект недвижимости (дома, квартира, промобъект)
Может посмотреть свои заказы
Остальной функционал в разработке
Так же в футере есть быстрые кнопки перехода в личный кабинет или каталог

Если недвижимость не добавлена переход на страницу виджета создания недвижимости по кнопке Добавить недвижимость 
(ссылка)

Если недвижимость уже создавалась отображается ее список в виде слайдера
