# Каталог

:::info Доступ
Доступно для: **всех авторизованных пользователей**
:::


Каталог товаров представляет собой дерево основных папок и вложенных вхождящих в них подкаталогов, 
каждая из которых может иметь свои товары и услуги по ним. 
Полностью интегрирован из каталога товаров и услуг ERP (англ. Enterprise Resource Planning, «планирование ресурсов предприятия») системы <Link to="https://www.moysklad.ru">МойСклад</Link>

На Основной странице каталога категории (группы) товаров высшего порядка

В настоящий момент в нашей системе есть одна такая категория "Очистка воды"

На странице каталога, как и на странице <Link to="sub-catalog">подкаталога</Link> и <Link to="product">товара и услуг по нему</Link> есть основной функционал навигации 

- в нижней части экрана
    - Личный кабинет
    - Каталог


- поиск товаров, открывается окно с поиском по товарам и их описаниям


- виджет корзины, при нажатии на которой можно попасть на страницу корзины заказа
