# Заказ

:::info Доступ
Доступно для: **всех авторизованных пользователей**
:::


Действия и состав копонентов страницы заказа различаются от роли пользователя. 

Все действия по изменению статуса, датыб исполнителя, склада заказа сопровождаются соответствующими сообщениям в телеграм

## Клиент


- Карта и адрес объекта недвижимости для услуг или установки товара
- Запланированная или желаемая дата заказа(если нет исполнителя по адресу заказа)
- Возможные статусы для клиента
- Исполнитель, если выбран при оформлении заказа или назначен куратором
