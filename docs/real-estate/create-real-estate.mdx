# Создание

:::info Доступ
Доступно для: **всех авторизованных пользователей**
:::

Страница создания объекта недвижимости представляет собой визард мастер пошагового создания сущности объекта недвижимости. 

#### Шаг 1. Вид объекта недвижимости и его источник воды. 

- Выберите тип (дом, квартира, промышленный объект)
- Выберите количество жильцов, или в случае пром объекта рабочих предприятия. Передвиньте ползунок на нужное кол-во
- Выберите источник воды (скважина, колодец, водоем, водопровод)
- Если скважина или колодец - передвиньте ползунок на нужноу глубину
